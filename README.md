# AI Movie Recommendation System

An intelligent Movie Recommendation System that suggests personalized movies based on user preferences, ratings, and content similarity.
Built using Machine Learning techniques and enhanced with a modern interactive UI using Streamlit.

---

##  Features

-  Personalized Movie Recommendations  
-  User Rating System  
-  AI-Based Search (Natural Language Search)  
-  Movie Trailers Integration (YouTube API)  
-  Movie Posters (OMDb API)  
-  User Login System  
-  User Preference Dashboard  
-  Fast Content-Based Filtering  

---

##  Tech Stack

- Python   
- Scikit-learn (ML Algorithms)  
- Pandas & NumPy (Data Processing)  
- Streamlit (Frontend UI)  
- OMDb API (Movie Posters)  
- YouTube Data API (Trailers)  

---
**How It Works**

- User selects or searches a movie
- System converts movie features into vectors
- Uses cosine similarity to find similar movies
- Enhances recommendations using user ratings
- Displays results with posters & trailers

**Recommendation Techniques**

- Content-Based Filtering
- Cosine Similarity
- TF-IDF Vectorization (AI Search)

**Sample Features**

- Search: "romantic comedy movies"
- Get recommendations instantly
- Watch trailers directly
- Save ratings and personalize results
